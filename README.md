# Page Analyzer

This is a demo application designed for checking the SEO suitability of a website.

## Deploy

Deploy on [Render.com](https://java-project-72-qdps.onrender.com/)

## Setup

```bash
make build
```
## Run
```bash
make run
```
## Run checkstyle
```bash
make lint
```
## Run tests
```bash
make test
```
